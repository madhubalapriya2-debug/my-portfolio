Flood Susceptibility Analysis in Khagaria District, Bihar

Project One Pager

🎯 Objective

The present study aims to identify flood-prone areas in the Khagaria district and propose appropriate flood mitigation strategies by employing a multicriteria decision-making approach inside a GIS framework, utilising remotely sensed data.

⚙️ Methodology

  • Data collection from USGS, CRU, Copernicus, GSI.
  • Data cleaning and preprocessing
  • Reclasifying the layers and weighted overlay analysis
  • Flood Vulnerability mapping
  • Spatial Joining of building layer with flood layer with python script
  • Final visualization through WEB Mapping
